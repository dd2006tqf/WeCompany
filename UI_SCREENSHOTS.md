## 按钮图标说明 (Button Icons)

使用 IconFont 字体图标：

- **挂断**: 0xe60e (电话挂断图标)
- **接听**: 0xe615 (电话接听图标)
- **静音**: 0xe641 (麦克风图标)
- **取消静音**: 0xe640 (麦克风静音图标)
- **视频开**: 0xe647 (摄像头图标)
- **视频关**: 0xe646 (摄像头关闭图标)

- 接听按钮: #00CC44 → #00AA33
- 拒接/挂断: #FF4444 → #FF2222

### 输入框 Focus 效果
- 边框: #CCCCCC → #1485EE

### 状态提示
- 成功: 绿色文字 #00AA00
- 错误: 红色文字 #FF0000
- 信息: 蓝色文字 #1485EE

## 使用流程 (Usage Flow)

### 登录流程
1. 启动应用 → 显示登录对话框
2. 输入用户名密码
3. 点击"登录"或"注册新账号"
4. 连接服务器并验证
5. 显示状态信息
6. 成功后进入主界面

### 拨打电话流程
1. 在联系人列表选择用户
2. 点击通话按钮（音频/视频）
3. 显示拨打电话对话框
4. 等待对方响应
5. 接通后转换为通话中界面
6. 结束通话

### 接听电话流程
1. 收到来电通知
2. 显示来电对话框
3. 用户选择接听或拒接
4. 接听后显示通话中界面
5. 结束通话

## 响应式布局 (Responsive Layout)

所有对话框使用固定尺寸，确保在不同屏幕上的一致性：
- 登录对话框: 400x500px
- 来电/呼出: 400x550px
- 通话中: 600x700px
